다중 언어 팩을 사용하는 사용자를 위한 중요 정보

적용 대상:
 - 다중 언어 팩을 설치하고 기본 템플릿 설치 위치(C:\Users\Public\Documents\Autodesk\Inventor release year\Templates)를 유지하는 모든 사용자.
 - C:\Users\Public\Documents\Autodesk\Inventor release year\Templates 폴더에 사용자 템플릿을 저장하고 사용하는 모든 사용자.

Inventor에는 여러 릴리스에서 지원되는 언어 팩이 포함되어 있습니다. 따라서 단일 Inventor 설치에 여러 언어를 설치할 수 있고 시작 메뉴 바로 가기(/language 매개변수 사용)를 통해 언어 간을 전환할 수 있습니다.  

언어 팩은 현지화된 템플릿 파일을 설치하지만 Inventor의 다른 언어를 실행해도 기본적으로 사용된 템플릿은 변경되지 않습니다. 

Inventor 2020년에서는 모든 언어 팩이 동일하게 설치되고 처리됩니다. 언어를 변경할 때 템플릿 파일 경로가 업데이트되므로 템플릿을 상위 폴더에 더 이상 복사하지 않아도 됩니다. 따라서 기본 템플릿이든 사용자 템플릿이든 템플릿의 사용 및 유지관리가 단순화됩니다.
